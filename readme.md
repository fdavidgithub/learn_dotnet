## Install

2. dotNet Core init
```console
export DOTNET_ROOT=/opt/dotnet
dotnet new webapi
dotnet tool install --global dotnet-ef
```

3. dotNet runtime
```console
dotnet run
```

4. Browser URL
localhost:5000

5. Swagger on Firefox
https://addons.mozilla.org/en-US/firefox/addon/access-control-allow-origin/
